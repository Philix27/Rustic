# Introduction
### Introduction and Project Setup

This project is centered around building a backend service for a task management system
in rust. It will serve as the backend serve to an existing task management & chat application.
We will begin by setting up our project structure. 

Goals
- A durable and easy scalable microservices architecture that handles all the core part of the application.

Core Services
- Notification service
- API Gateway
- Log service
- User and Auth service
- Chat Service
